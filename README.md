## Prerequisites

Have `npm` (or `pnpm`), `pip` and `cargo` installed.

If using `npm`:

```bash
alias pnpm="npm"
```

Else:

```bash
pnpm install -g neovim
python3 -m pip install --user pynvim
cargo install fd-find ripgrep
```

## Installation

```bash
git clone https://github.com/willswats/neovim-config.git ~/.config/nvim
```
